- Cacao segregator

Classifications based on color:
- Criollo       [white]
- Forastero     [dark-purple]
- Trinotario    [combination of Criollo and Forastero]

Process:
- Cut cacao into half.
- Insert into 5 cutted beans inside segregator machine.
- Capture images of cacao beans.
- Resize the image to match the expected size of the model.
- Classify beans based on color.

- If color is white and color is dark-purple: open Trinitario window.
- Else If color white: open Criollo bean window
- Else if color dark-purple: open Forastero been window
- Else don't open window [other window for unknown been or re-check]
